# 智阅 AI v0.1.1 发布准备

## 发布日期
2026-01-28

## 版本说明
这是智阅 AI 的首个公开发布版本，包含核心的文本简化和术语解释功能。

---

## ✅ 已实现功能

### 1. 核心功能
- **文本简化**：使用 AI 将复杂文本转换为易懂的表达
  - 支持多种 AI 服务商（OpenAI GPT-4, Claude, Moonshot）
  - 智能提示词优化，基于"智阅AI"核心理念
  - 关键词高亮显示
  - 元数据展示（领域、置信度、处理时间、AI 模型）
  
- **术语解释**：对选中的专业术语进行详细解释
  - 分层解释（是什么 → 为什么 → 怎么用）
  - 使用类比和日常例子
  - 关键词高亮显示
  
- **文本复制**：快速复制选中的文本到剪贴板

### 2. 用户界面
- **划词工具栏**：
  - 毛玻璃效果设计
  - 三个按钮：简化、解释、复制
  - 加载状态显示
  - 智能定位（避免超出屏幕）
  
- **结果展示卡片**：
  - 毛玻璃效果设计
  - 可拖拽移动
  - 可展开/收起
  - 显示原文和处理结果
  - 关键词内联高亮（低调风格）
  - 元数据信息（4 个卡片：领域、置信度、处理时间、AI 模型）
  - 固定底部操作按钮（复制、收藏、来源）
  - 加载动画（Spinner）
  - 错误提示
  
- **浅色/暗黑模式**：自动适配系统主题

### 3. 设置页面
- **AI 配置**：
  - 支持 4 个 AI 服务商配置
    - OpenAI GPT-4 ✅
    - Anthropic Claude ✅
    - Moonshot (Kimi) ✅
    - 百度文心一言 ⏳（UI 已完成，功能待实现）
  - API 密钥加密存储（AES-256-GCM）
  - 选择默认服务商
  - 密钥显示/隐藏切换
  
- **界面设置**：
  - 主题选择（浅色/暗黑/自动）
  - 语言选择（简体中文/English）
  - 功能开关（自动翻译、知识库）
  
- **数据管理**：
  - 数据导出
  - 清除所有数据
  - 恢复默认设置

### 4. 弹出窗口（Popup）
- 搜索知识库（UI 已完成，功能使用 Mock 数据）
- Token 使用统计（UI 已完成，功能使用 Mock 数据）
- 快速主题切换
- 快速访问设置页

### 5. 技术实现
- Shadow DOM 隔离，避免样式冲突
- CSP 合规（Manifest V3）
- 代码块选择优化（多层回退策略）
- 性能优化（防抖、缓存）
- 错误处理和重试机制
